## Week 2 – GAN Implementation (Lab 02)

This lab demonstrates the implementation of a Generative Adversarial Network (GAN) using TensorFlow.

### Datasets
- MNIST
- Fashion-MNIST

### Files
- `gen_ai_lab_02.py` – GAN training and image generation code
- `generated_samples_*.zip` – Images generated during training (epoch-wise)
- `final_generated_images_*.zip` – Final generated images after training

Due to GitHub upload limits, generated images are provided as ZIP archives.
