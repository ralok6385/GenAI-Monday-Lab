# Week 3 – Variational Autoencoder (VAE)

## Course
CSET-419 – Introduction to Generative AI

## Objective
To implement a Variational Autoencoder (VAE) to learn latent representations and generate new images.

## Dataset
MNIST / Fashion-MNIST

## Description
This lab demonstrates the implementation of a Variational Autoencoder using TensorFlow.
The encoder learns the mean and variance of the latent space, and the decoder reconstructs
images using the reparameterization trick.

## Files
- lab3.py – VAE implementation
- GenAI-Lab-Week3.docx – Lab assignment
- vertopal.com_Lab3.pdf – PDF submission

## Output
- Reconstructed images
- Generated images from latent space

## Conclusion
The VAE successfully learns a probabilistic latent space and generates meaningful samples.
